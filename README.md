# Skill Swap Platform

## 📌 Overview
Skill Swap Platform is a full-stack web application that allows users to exchange skills by connecting with others who have matching interests.

Users can:

Add their skills
View other users' skills
Find matching skills for collaboration

## 🚀 Features
🔐 User Registration & Login
➕ Add Skills
🔍 View Skills
🤝 Skill Matching System
💾 Database Integration (MySQL)

## 🛠 Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js
Database: MySQL

📂 Project Structure
server.js → Backend logic
index.html → Home page
login.html → Login page
register.html → Registration page
add.html → Add skills
view.html → View skills

## ⚙️ Setup Instructions
1. Clone the repository
2. Install dependencies:
   npm install

3. Configure database in server.js

4. Start server:
   node server.js

5. Open browser:
   http://localhost:3000
