# Skill Swap Platform

## 📌 Overview
Skill Swap Platform is a full-stack web application that allows users to exchange skills by teaching one skill and learning another.

## 🚀 Features
- User skill registration
- View all users
- Skill matching logic
- Connect via email
- Responsive UI with Bootstrap

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Node.js (Express)
- Database: MySQL

## ⚙️ Setup Instructions
1. Install dependencies:
   npm install

2. Configure database in server.js

3. Start server:
   node server.js

4. Open browser:
   http://localhost:3000

## ⚠️ Note
Update MySQL password in server.js before running the project.
